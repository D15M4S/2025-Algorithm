#### Commit Convention

- [문제 출처(플랫폼)] 문제이름, / 난이도
- 플랫폼 작성법 통일:
  - [BOJ] - 백준
  - [PGS] - 프로그래머스
  - [LTC] - 리트코드
  - [CFS] - 코드포스
  - [SEA] - 삼성SW Expert Academy
  - [ETC] - 그외

- {풀이에 쓴 시간}/80m
- {풀이에 사용한 알고리즘 및 기법}
- {본인의 한계점, 무엇을 더 공부해야 할지?}
